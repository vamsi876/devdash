DevDash

12 developer utilities in your macOS menubar. Format, decode, generate, convert — all one click away.

Terminal
$ pip install devdash
Installing collected packages: devdash
$ devdash
✓ DevDash running in menubar
# 12 tools ready — click the menubar icon
Tools

12 tools. Zero context-switching.

Everything you reach for daily, accessible from a single menubar icon.

{ }

JSON Formatter

Pretty-print, minify, and validate JSON with syntax highlighting and error reporting.

🔑

JWT Decoder

Decode JWT headers and payloads instantly. Verify expiration and inspect claims.

🛡

UUID / ULID Generator

Generate v4 UUIDs and ULIDs in bulk. Copy with one click, sorted by creation time.

Base64 Encode/Decode

Encode and decode Base64 strings. Handles text and detects binary content.

Timestamp Converter

🔗

Workflow

Smart Clipboard

DevDash watches your clipboard. Copy a JWT and the decoder opens automatically. Copy JSON and it formats instantly. No searching, no switching — the right tool appears when you need it.

1

Copy anything

Copy a JWT, JSON blob, Unix timestamp, or Base64 string to your clipboard.

▼
2

Auto-detect

DevDash identifies the content type and routes to the right tool instantly.

▼
3

Result ready

Formatted output is one click from your clipboard. Paste and move on.

Get started

Install in seconds

Requires Python 3.11+ and macOS.

$ pip install devdash

Then run devdash to launch. Use devdash --help for options.
